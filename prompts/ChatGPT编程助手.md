# 💻 ChatGPT 编程助手提示词

> 让AI帮你写代码、调试、优化，提高开发效率。

---

## 一、代码生成

| 场景 | 提示词模板 | 示例 |
|------|-----------|------|
| 生成函数 | 用[语言]写一个函数，功能是...，输入是...，输出是...。请包含异常处理。 | 用Python写一个函数，功能是计算列表平均值，输入是数字列表，输出是浮点数。请包含异常处理。 |
| 生成类 | 用[语言]创建一个[类名]类，包含属性...和方法... | 用JavaScript写一个User类，包含name和email属性，以及一个验证邮箱格式的方法validateEmail()。 |
| 生成API | 用[框架]写一个RESTful API，路由...，方法[GET/POST]，接收参数...，返回... | 用Express写一个POST接口`/register`，接收JSON格式的username、password、email，返回用户ID和201状态码。 |
| 生成数据库操作 | 用[语言/ORM]写一个函数，实现[增删改查]操作，表名...，条件... | 用Python + SQLAlchemy写一个函数，根据user_id查询User表，返回用户名和邮箱。 |
| 生成正则表达式 | 帮我写一个正则表达式，用于匹配[某种模式] | 帮我写一个正则表达式，用于匹配中国大陆11位手机号码（以1开头，第二位3-9）。 |

## 二、代码优化

| 场景 | 提示词模板 | 要点 |
|------|-----------|------|
| 性能优化 | 请优化以下代码的性能，并说明优化思路：<br>（粘贴代码） | 说明当前复杂度瓶颈（如O(n²)），给出优化后代码和复杂度分析 |
| 代码审查 | 请审查以下代码的问题，重点关注安全性、可读性、边界条件：<br>（粘贴代码） | 输出问题列表 + 修改建议 + 重构后的代码 |
| 重构建议 | 这段代码太长了，请拆分成多个小函数，保持逻辑不变：<br>（粘贴代码） | 保持单一职责，函数命名清晰，不改变外部接口 |
| 消除重复 | 以下代码中存在重复逻辑，请帮我抽取公共部分：<br>（粘贴代码） | 使用DRY原则，提取公共函数或基类 |
| 内存优化 | 这段代码处理大文件/大数据时内存占用过高，请优化：<br>（粘贴代码） | 建议使用生成器、流式处理、分块加载或数据库游标 |

## 三、调试帮助

| 场景 | 提示词模板 | 示例 |
|------|-----------|------|
| 报错分析 | 我遇到了一个错误：[粘贴错误信息]<br>我的代码是：[粘贴相关代码]<br>请帮我分析可能的原因和修复方案。 | `IndexError: list index out of range`<br>代码：`for i in range(len(lst)+1): print(lst[i])`<br>分析：循环多了一次，应改为`range(len(lst))` |
| 添加日志 | 为以下代码添加详细的日志（使用[日志库]），帮助定位问题：<br>（粘贴代码） | 用Python的logging模块，记录函数入口、关键变量值、异常信息 |
| 单元测试生成 | 为以下函数/类生成单元测试，覆盖正常值、边界值和异常情况：<br>（粘贴代码） | 针对`divide(a,b)`函数，测试b=0、a=0、负数、浮点数等情况 |
| 模拟Bug复现 | 这段代码在输入[具体值]时输出[错误结果]，预期输出[正确结果]。请找出bug：<br>（粘贴代码） | 排序函数对`[5,5,1]`排序返回`[1,5]`丢失了一个5，请修复 |
| 并发问题分析 | 以下多线程/异步代码偶尔出现死锁/竞态条件，请分析并给出解决方案：<br>（粘贴代码） | 使用threading.Lock但未正确释放，建议使用`with lock:`上下文管理器 |

## 四、进阶技巧

| 技巧 | 说明 | 示例提示词 |
|------|------|-----------|
| 指定输出格式 | 要求代码包含注释、类型注解、docstring | “请为函数添加Google风格的docstring，以及类型注解” |
| 分步解释 | 先让AI解释思路，再给出代码 | “请先解释解决这个问题的算法思路，然后写出Python代码” |
| 角色扮演 | 让AI扮演资深工程师进行代码评审 | “请你作为一位资深后端工程师，审查以下代码的可维护性” |
| 对比方案 | 要求提供多种解决方案并比较 | “给出两种实现方式，并说明各自的优缺点和适用场景” |
| 限定环境版本 | 指定Python 3.8+ / ES6 / React 18等 | “请使用Python 3.8+的walrus运算符简化代码” |

---

> 🧠 **小贴士**：把代码粘贴到ChatGPT时，用三个反引号包裹并标明语言（如 ```python），输出更准确。遇到复杂问题，加上“请一步一步思考”可大幅提升成功率。
